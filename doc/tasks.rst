Documentation: Tasks
**************************

.. automodule:: prkng

The Tasks system uses Py-RQ and RQ-Scheduler to accomplish tasks based on initiating events as well as certain dates and times. Tasks include obtaining updated data for carsharing services and sending push notifications for snow removal.
